Add a survival game mode, where you try to survive specific challenges or waves of challenges, and have options for different numbers of players.
Create leaderboards for decks that attempt different challenges as well as their highest scores.
Rollback option to delete the previous roll.
Set it up so you start each game with a random enchant world, and add in a setting that will be checked by default that can toggle this option.
Add in details from mtg sites like the recent banned/restricted lists, what all the random new abilities that come out each month do, etc.
Create specific rules for situational rolls. Death roll? Higher chance of life gain. Rolls for the first 3 turns could have low intensity.
